Co patří mezi charakteristiky IPv6?

  • bezpečnostní mechanismy přímo v IP
  • zpětná kompatibilita s IPv4
  • bohatý adresní prostor
  • podpora mobilních zařízení

Označte pravdivé výroky o VLAN:

  • Vytváří doménu všesměrového vysílání.
  • Vytváří logický segment LAN, kde uzly jsou připojené k různým fyzickým segmentům sítě.
  • Technologie vznikla v polovině 80. let 20. století.
  • Umožňuje lepší administraci počítačových sítí.
  • Členění sítě probíhá na síťové vrstvě.

Co nepatří do SIP architektury:

  • registrar server
  • proxy server
  • MCU
  • location server
  • gatekeeper

Adresa ::1/128 v IPv6 představuje:

  • libovolnou adresu
  • lokální adresu loopback
  • unikátní individuální lokální adresu
  • individuální lokální linkovou adresu

Označte pravdivé výroky o globálním katalogu v Active Directory:

  • Snižuje množství replikací.
  • Obsahuje kompletní repliky informací obsažených v ostatních doménách.
  • Nedoporučuje se dělat repliky na další doménové řadiče.
  • >Globálním katalogem je standardně 1. nainstalovaný řadič domény.

Leaderem na trhu virtualizace je firma:

  • VMware
  • Red Hat
  • Microsoft
  • Citrix

Jaké jsou možnosti zajištění kvality služeb (QoS):

  • implementace VLAN
  • packet shaper
  • předimenzování konektivity
  • priority

Co neplatí o H.323 standardu:

  • RTCP zajišťuje přenos multimediálních dat
  • RTP zajišťuje přenos řídicích informací
  • komplexní standard ze světa spojů
  • uživatelé se mohou přemísťovat
  • gatekeeper zprostředkovává vyhledání volaného

Označte pravdivé výroky o LDAP protokolu:

  • dokonalejší verze standardu X.500
  • optimalizace uložených dat pro vyhledávání a čtení
  • Každý záznam má hodnoty označované jako atributy.
  • data ukládána ve dvourozměrném poli

Označte pravdivé výroky o Windows Serveru:

  • aktuální verze je Windows Server 2022 R2
  • v Datacenter edici možnost instalovat na libovolný počet virtuálních serverů bez specifikace počtu CPU jader
  • serverová technologie vyvíjená od 90. let minulého století
  • široká možnost nasazení – doménový řadič, stand-alone server, web server

Označte pravdivé výroky o Remote Desktop Services (Terminal Services):

  • Pro trvalý provoz nejsou nutné dodatečné licence.
  • Kromě obrazu dokáže přenášet soubory, clipboard a USB periferie.
  • Pro připojení je potřeba klient, který není standardně součástí Windows.
  • protokol není zpětně kompatibilní.
  • Nepodporuje šifrování přenášených dat.

Označte pravdivé výroky o NFS:

  • v prostředí Windows se jedná o nejpoužívanější protokol
  • nastavení sdílených adresářů je v /etc/exports
  • protokol pro vzdálený přístup k souborům přes počítačovou síť
  • funguje výhradně nad transportním protokolem TCP

Označte nepravdivý výrok o gridovém clusteru:

  • často tvořen z heterogenních počítačů, různě geograficky distribuovaných
  • optimalizován pro řešení úloh, které potřebují sdílet data během výpočtu
  • cluster zaměřený na celkovou propustnost systému

Co zajišťuje brána u H.323?

  • nemění signalizační protokoly
  • konverze na úrovni přenosu dat (Ethernet, ATM, …)
  • převod mezi různě kódovaným hlasem
  • konverze mezi různými systémy a protokoly

Úroveň virtualizace nazývaná emulace / simulace znamená spuštění:

  • spuštění libovolného OS virtuálního počítače pro stejnou procesorovou platformu hostitele
  • spuštění virtuálního počítače se zcela odlišnou procesorovou platformou než je hostitel
  • pouze stejného operačního systému virtuálního počítače jako je systém hostitele

Označte pravdivé výroky o IIS:

  • IIS 10 přináší podporu wildcard notace.
  • IIS 10 přináší podporu Nano Serverů.
  • Jedná se o nejpoužívanější webový server na internetu.
  • IIS 9 je evolucí předchozí verze.
  • IIS 10 nepodporuje HTTP/2.

Označte pravdivá tvrzení o standardu 802.1Q:

  • Dává uživateli možnost definice až 212 VLAN
  • Rozšiřuje IP paket o 4 byty
  • Dává uživateli možnost definice až 4094 VLAN
  • >Rozšiřuje ethernetový rámec o 2 byte

Členství ve VLAN lze definovat podle:

  • podle síťových adres uzlů
  • podle výrobce síťového hardwaru
  • MAC adres uzlů
  • podle portu
  • podle skupinového vysílání

Mezi nejdůležitější vlastnosti ovlivňující síťové aplikace patří:

  • ztrátovost paketů
  • rychlost disku
  • šířka pásma
  • rozptyl zpoždění

Označte pravdivé výroky o standardu 1000Base-X u gigabitového ethernetu:

  • podpora pro stíněné kabely „twinax“ STP (Shielded Twisted Pair)
  • podpora pro nestíněné kabely UTP (Unshielded Twisted Pair)
  • podpora 850 nm laser pro mnohavidová vlákna
  • podpora 1310 nm laser pro jednovidová a mnohavidová vlákna
